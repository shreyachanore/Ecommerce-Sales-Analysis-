# E-Commerce SQL Data Analysis Project

## Overview
This project focuses on analyzing a real-world E-commerce dataset using SQL to extract meaningful business insights. The analysis helps understand customer behavior, sales performance, product trends, and return patterns.

---

## Objectives
- Analyze customer purchasing behavior
- Evaluate sales and order performance
- Identify top-selling products
- Understand return trends
- Practice real-world SQL querying

---

## Tools & Technologies
- SQL (MySQL)
- MySQL Workbench
- Microsoft Excel / CSV
- Git & GitHub

---

## Dataset Description
The project uses multiple CSV files representing an E-commerce database:

- customers_final.csv – Customer details
- orders_final.csv – Order information
- order_items_final.csv – Order and product mapping
- products_final.csv – Product details
- returns_final.csv – Returned orders data

All datasets were imported into MySQL for relational analysis.

---

## SQL Concepts Used
- Database and table creation
- Primary and foreign keys
- SELECT, WHERE, ORDER BY
- GROUP BY and HAVING
- Aggregate functions (COUNT, SUM, AVG)
- Joins (INNER JOIN, multi-table joins)
- Subqueries
- Business-oriented analytical queries

---

## Key Analysis Performed
- Total customers and orders analysis
- Revenue and sales performance analysis
- Product-wise sales insights
- High-value customer identification
- Return rate and trend analysis

---

## Repository Structure
- Ecommerce.sql – SQL queries used for analysis
- *.csv files – Raw dataset files
- README.md – Project documentation

---

## How to Run the Project
1. Import all CSV files into MySQL Workbench
2. Run the queries from Ecommerce.sql
3. Analyze the output to derive insights
